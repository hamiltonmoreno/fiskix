# Security Policy for Reporting Vulnerabilities

## Reporting a Vulnerability
If you discover a security vulnerability in this project, we encourage you to report it to us as soon as possible. Please follow these steps:

1. **Do not disclose the vulnerability publicly until it has been patched.**
2. **Contact us via email:** [email@example.com](mailto:email@example.com)
3. Provide a detailed description of the vulnerability, including steps to reproduce it and any relevant links.

## Responsible Disclosure
We take security seriously and will work to address any vulnerabilities promptly. We will make every effort to fix the vulnerability and notify you before making any public disclosures.

## Acknowledgments
If you report a vulnerability, you will be acknowledged in our project’s documentation, as a token of our appreciation for helping us improve security.
