Secção 3 — Página de Alertas

Redesign da tabela de alertas com shadcn/ui. Qual o padrão de interacção preferido?

A

Tabela + Modal (actual)

Clicar numa linha abre modal com detalhe. Simples, mas o modal corta o contexto da lista.

B

Tabela + Sheet lateral (slide-over)

Clicar abre um painel deslizante à direita. Lista fica visível atrás. Padrão moderno SaaS.

C

Lista expandível inline (accordion)

Clicar expande a linha. Sem navegação extra. Bom para revisão rápida de muitos alertas.
